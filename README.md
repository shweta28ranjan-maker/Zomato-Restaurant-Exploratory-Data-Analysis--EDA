# Zomato Restaurant Data Analysis using Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on Zomato restaurant data using Python. The objective is to analyze restaurant trends, customer preferences, pricing patterns, cuisines, ratings, and online ordering behavior to generate meaningful business insights.

The project demonstrates practical data analysis skills including data cleaning, preprocessing, visualization, and business interpretation using real-world restaurant data.

---

# Objectives

* Analyze restaurant data and customer behavior trends
* Identify the most popular restaurant types and cuisines
* Study the impact of online ordering on customer ratings
* Explore restaurant pricing distribution
* Identify top restaurant locations based on density and engagement
* Generate business insights through data visualization

---

# Dataset Information

The dataset contains information about restaurants listed on Zomato, including:

* Restaurant Name
* Location
* Restaurant Type
* Cuisines
* Ratings
* Votes
* Approximate Cost for Two People
* Online Ordering Availability
* Table Booking Availability

# Key Insights

* Casual Dining and Quick Bites are the most common restaurant types.
* Online ordering availability does not strongly influence customer ratings.
* Most restaurants maintain ratings between 3.5 and 4.5.
* North Indian cuisine is the most popular cuisine category.
* Restaurant density is highest in commercial urban locations.
* Affordable and mid-range restaurants dominate the market.
